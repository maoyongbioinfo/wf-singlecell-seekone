# wf-singlecell-seekone
The software was developed by YongMao (Wuhan Benagen Tech Solutions Company Limited) based on the wf-singlecell v3.3.4 (https://github.com/epi2me-labs/wf-single-cell), This allows the
software to expand its supporing analysis for SeekOne single-cell platform following by nanopore long read sequencing.  Notice. the origin fucntion for 10x singlecell or Visium HD full
length are also kept. The software is public for all reserchers.
